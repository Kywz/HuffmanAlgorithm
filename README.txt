Description:
This program takes text, that is inputed by user (left-top text box), creates a frequencys list and a tree list (right-top text box), 
with which programm encodes inputed text into binary Huffman code (left-bottom text box). Then, the created code, 
is decoded and transferred into text form again (left-top text box).
